Guía de Calificación
{% set cod = evaluacion.cuestionario.codigo %} {% if cod in ('PHQ-2', 'PHQ-9') %}
PHQ — Patient Health Questionnaire

Cálculo del puntaje: Sumar los valores de todas las respuestas (0–3 por pregunta).

Escala de respuesta:

  • 0 = Para nada
  • 1 = Varios días
  • 2 = Más de la mitad de los días
  • 3 = Casi todos los días
{% if cod == 'PHQ-2' %}

Punto de corte: Puntaje ≥ 3 indica tamizaje positivo para depresión; se recomienda aplicar PHQ-9.

{% else %}

Pregunta 9 (ideación suicida): Cualquier respuesta ≥ 1 genera alerta crítica independientemente del puntaje total.

Puntaje máximo: 27 puntos (9 preguntas × 3 puntos).

{% endif %} {% elif cod == 'AUDIT' %}
AUDIT — Alcohol Use Disorders Identification Test (OMS)

Cálculo del puntaje: Sumar las respuestas de las 10 preguntas.

  • Preguntas 1–8: valores 0, 1, 2, 3, 4
  • Preguntas 9–10: valores 0, 2, 4

Puntaje máximo: 40 puntos.

{% elif cod == 'ASSIST' %}
ASSIST — Alcohol, Smoking and Substance Involvement Screening Test (OMS)

Q1: Pregunta de cribado (uso en la vida). Si la respuesta es "No" para una sustancia, su SSIS = 0.

SSIS por sustancia: Suma de respuestas Q2 + Q3 + Q4 + Q5 + Q6 + Q7 para cada sustancia reportada en Q1.

Q8 — Uso inyectado: Pregunta crítica. Cualquier respuesta positiva genera alerta; "Sí (últimos 3 meses)" = riesgo máximo.

Puntaje máximo SSIS: 39 puntos por sustancia.

{% elif cod in ('IDARE', 'IDERE') %}
IDARE/IDERE — Inventario de Ansiedad Estado-Rasgo

Cálculo: Sumar las respuestas. Algunos ítems tienen puntuación inversa según el instrumento original.

Puntaje total: Suma directa de los valores seleccionados en esta aplicación.

{% elif cod == 'ADNM-8' %}
ADNM-8 — Adjustment Disorder New Module (versión breve)

Cálculo: Los ítems 1–4 evalúan preocupación e interferencia (0–3 cada uno).

Punto de corte: ≥ 8 indica indicador positivo para trastorno adaptativo.

{% elif cod == 'ADNM-20' %}
ADNM-20 — Adjustment Disorder New Module (versión completa)

Cálculo: Suma de todos los ítems. Evalúa presencia e interferencia de síntomas de trastorno adaptativo tras un evento estresante.

Puntaje máximo: {{ evaluacion.cuestionario.puntaje_maximo }} puntos.

{% else %}

Puntaje total: Suma de todos los valores de respuesta seleccionados. Puntaje máximo: {{ evaluacion.cuestionario.puntaje_maximo }}.

{% endif %}
